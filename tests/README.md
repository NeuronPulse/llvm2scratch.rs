# Integration tests (Python vs Rust)

This directory contains differential/integration tests that compare the Python
reference implementation against the Rust reimplementation.

| Script | What it tests | Input |
|---|---|---|
| `diff_test.py` | Full `.sb3` project structure equivalence | `examples/input/*.ll` |
| `scratchblocks_diff_test.py` | Readable scratchblocks text output | `examples/input/*.ll` |
| `scratchblocks_stress_diff_test.py` | Readable scratchblocks text output on generated stress programs | `/tmp/llvm_stress_main/ll/*.ll` (generated by `scripts/generate_complex_programs.sh`) |

Run all differential tests at once with `bash scripts/run_diff_tests.sh`.
